## CTA Templates by Content Type

**For Awareness Articles:**
```
"Thought you'd find this relevant: [Article Title]. [One-sentence hook]. Worth 5 minutes if you're [pain point]."
```

**For Consideration Articles:**
```
"If you're exploring [topic], this breaks down [specific insight]. Might be useful for [their situation]."
```

**For Decision/Comparison Articles:**
```
"If you're evaluating [competitor/category], this comparison might save you time. Breaks down [key differentiator]."
```

**For Case Study References:**
```
"[Company similar to theirs] achieved [specific metric] with [approach]. Happy to share details on how they did it."
```
