# Inkeep GTM Funnel Optimization: Master Report

**Generated:** March 10, 2026
**Status:** Actionable Recommendations
**Goal:** Increase demo bookings from ~15/month to 40+/month

---

## Executive Summary

Inkeep's GTM funnel has significant conversion leaks at every stage. Analysis of website analytics, outreach campaigns, and pipeline data reveals that **the primary issue is not traffic—it's conversion**. The website receives 21,000+ homepage sessions and 3,000+ blog sessions per quarter, but converts at 10-15x below industry benchmarks.

**Key Finding:** The blog-to-demo funnel is completely broken (0% conversion on high-traffic posts), while case studies convert at 24% but receive minimal traffic. Fixing content placement and CTAs—not generating more traffic—is the highest-leverage opportunity.

### Current vs Target Performance

| Metric | Current (90 days) | Target (90 days) | Gap |
|--------|-------------------|------------------|-----|
| Homepage → Demo | 0.20% | 1.5% | 7.5x |
| Blog → Demo | ~0% | 2% | ∞ |
| Demo Page Submission | 3.79% | 10% | 2.6x |
| Case Study Traffic | 37 visitors | 200 visitors | 5.4x |
| Direct Win Rate | 3.1% | 10% | 3.2x |
| **Total Demo Submissions** | **~44** | **~150** | **3.4x** |

## 1. Website Conversion Analysis

### Traffic Overview (Last 90 Days)

| Page | Sessions | Current CVR | Issue |
|------|----------|-------------|-------|
| Homepage | 21,033 | 0.20% | 10x below median |
| /demo | 1,723 | 3.79% | At median, below top performers |

| Use-case pages | 2,411 | 3-4% | Dual CTA confusion |

**Source:** [Website Performance Report (March 9)](inkeep-marketing-website-performance-report-2026-03-09.md)

### Critical Issues Identified

#### Issue #1: Blog-to-Demo Funnel is Broken

**Evidence:**
- Top blog posts have **0% conversion** to /demo
- /blog/technical-b2b-support-in-2026: 880 visitors, 0 conversions
- /blog/composer-vs-swe: 738 visitors, 0 conversions
- Blog CTAs only in header/footer—users never see them

**Root Cause:** No inline CTAs connecting article topics to product value.

**Impact:** ~3,000 monthly visitors with zero conversion

#### Issue #2: Homepage Conversion is 0.20%

**Evidence:**
- 16,755 homepage visitors in 90 days
- Only 33 clicked through to /demo
- Industry median: 2-3%

**Root Cause:**
- No specific metrics in hero
- Generic CTA language ("Get a Demo" vs "Book a 15-min Demo")
- Missing trust signals (G2 badges, security certifications)

#### Issue #3: Case Studies Under-Leveraged

**Evidence:**
- /case-studies converts at 24.32%—highest of any page
- Only 37 users saw case studies before /demo
- Not featured on homepage or blog posts

**Opportunity:** If 150+ users see case studies (4x current), that's 36+ additional demo clicks.

#### Issue #4: Demo Page Drop-off is 96%

**Evidence:**
- 1,160 users reached /demo
- Only 44 submitted (3.79%)
- Top performers achieve 8-12%

**Current Strengths:** 2-field form (optimal), embedded calendar

**Missing:** Testimonials with metrics near form, "what to expect" clarity

**Source:** [B2B Demo Conversion Optimization Research](b2b-saas-demo-conversion-optimization/REPORT.md)

1. **Blog CTAs in emails** — Users click but don't convert to demo
2. **Scaling too fast** — 3x volume dropped engagement quality
3. **Wrong content targets** — Developer articles attract wrong audience (<1% CVR)

## 4. Cold Email Funnel Issues

### The Broken Funnel

```
Current Flow:
Email Send → Blog Click → No Inline CTA → Bounce → Lost

                    ↓ Some users try to convert ↓

                CTA Render Bug → Dead Click → Frustration → Lost

                    ↓ Some reach demo page ↓

                "ENTERPRISE" Label → Self-disqualify → Lost

                    ↓ Some try to submit ↓

                "Work Email Required" → Friction → Lost
```

### Root Causes (from Session Recording Analysis)

1. **No inline CTAs on blog** — Users scroll, read, leave
2. **CTA rendering race condition** — Button loads after user clicks
3. **"ENTERPRISE" label on CTA** — Deters non-enterprise prospects
4. **Work email required on form** — Blocks mobile/personal email users
5. **Traffic-persona mismatch** — Developer articles (<1% CVR) vs CX articles (8-21% CVR)

**Source:** [Blog Click to Demo Analysis](../Email Campaigns/outreach_blog_analysis/Blog_Click_to_Demo_Analysis.md)

### Content That Converts vs Doesn't

| Content Type | Traffic | Demo CVR | Action |
|--------------|---------|----------|--------|
| Developer how-to articles | High | <1% | Deprioritize in emails |
| CX/Enterprise articles | Medium | 8-21% | Route outreach here |
| "vs." comparison articles | Medium | 12-33% | High priority |
| Case studies | Low | 24% | Feature prominently |

**Source:** [Blog to Demo Funnel Analysis](../Email Campaigns/outreach_blog_analysis/evidence/inkeep-blog-demo-funnel-analysis.md)
